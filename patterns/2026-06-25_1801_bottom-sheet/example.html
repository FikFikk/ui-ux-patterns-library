Pattern UI/UX Modern untuk Mobile-First Design

1. Action Sheet — Quick Actions

Pattern paling umum untuk menampilkan daftar aksi yang bisa dilakukan pengguna. Cocok untuk sharing, export, atau menu kontekstual.

2. Form Input — Data Entry

Gunakan bottom sheet untuk form input yang tidak perlu full-page navigation. User tetap melihat konteks halaman di background.

3. Detail View — Informasi Lengkap

Tampilkan detail atau informasi tambahan tanpa navigasi penuh. Cocok untuk preview, detail produk, atau info place.

💡 Tips Interaksi

  • Swipe Down: Geser bottom sheet ke bawah untuk menutup (mobile)
  • Tap Backdrop: Klik area gelap di luar sheet untuk menutup
  • Close Button: Klik tombol × di header
  • Keyboard: Tekan ESC untuk menutup sheet

Tentang Bottom Sheet

📱 Apa itu Bottom Sheet?

Bottom Sheet adalah komponen UI yang muncul dari bagian bawah layar, menampilkan konten atau aksi tambahan tanpa mengalihkan pengguna dari konteks saat ini.

✨ Keunggulan

  • Thumb-Friendly: Mudah dijangkau dengan ibu jari di mobile
  • Context Preservation: User tetap melihat halaman di background
  • Gesture-Driven: Swipe untuk close terasa natural
  • Progressive Disclosure: Tampilkan info bertahap

🎯 Digunakan Oleh

Google Maps (place details), Instagram (sharing), Spotify (now playing), Gmail (email actions), Tokopedia (filters)
