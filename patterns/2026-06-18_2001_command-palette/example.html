Tentang Command Palette

Command Palette adalah interface pattern yang menggabungkan pencarian, navigasi, dan aksi dalam satu dialog terpusat. Pattern ini sangat populer di aplikasi modern seperti VS Code, GitHub, Slack, dan Linear.

Fitur utama: Keyboard-first, fuzzy search, contextual actions, dan keyboard navigation yang intuitif.
