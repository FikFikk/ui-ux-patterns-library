Fitur Aksesibilitas & UX Interaktif (Untuk Developer)
  • Fokus Otomatis Maju-Mundur: Fokus berpindah secara cerdas ke input berikutnya setelah mengetik, atau ke input sebelumnya saat menekan tombol Backspace.
  • Pencegahan Lompatan Salah: Mengklik sembarang input kosong di tengah akan otomatis memfokuskan field kosong pertama.
  • Fungsi Paste Tangguh: Salin angka 6-digit (termasuk strip seperti 123-456 atau teks acak), paste di kotak manapun, kotak otomatis terisi berurutan tanpa menduplikasi kolom.
  • Navigasi Panah Keyboard: Gunakan panah ← dan → untuk berpindah kolom secara natural.
  • Screen Reader-Friendly: Setiap kolom input memiliki atribut aria-label eksplisit (misal: "Digit ke-1 dari 6"). Hasil status verifikasi dikumpulkan dalam kontainer dengan aria-live="polite".
  • Autofill One-Time-Code: Atribut autocomplete="one-time-code" diaktifkan pada field pertama untuk integrasi autofill SMS iOS/Android dan Chrome modern.
