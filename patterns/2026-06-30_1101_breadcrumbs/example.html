4. Mobile-First Back Button Mobile

Untuk mobile, tampilkan simple "back" link ke parent page. Lebih touch-friendly dan hemat ruang.

Di mobile, cukup tampilkan back link ke parent category

<!-- Mobile-first approach --> <nav aria-label="Breadcrumb"> <a href="/parent-page" class="back-link"> <svg>...</svg> Kembali ke Parent </a> </nav> <!-- Or responsive: show full on desktop, truncated on mobile --> @media (max-width: 768px) { .breadcrumb li:not(:last-child):not(:nth-last-child(2)) { display: none; } }

6. Schema.org Markup untuk SEO SEO

Tambahkan structured data agar Google menampilkan breadcrumbs di search results.

<!-- JSON-LD Schema Markup --> <script type="application/ld+json"> { "@context": "https://schema.org", "@type": "BreadcrumbList", "itemListElement": [ { "@type": "ListItem", "position": 1, "name": "Home", "item": "https://example.com" }, { "@type": "ListItem", "position": 2, "name": "Elektronik", "item": "https://example.com/electronics" }, { "@type": "ListItem", "position": 3, "name": "Smartphone", "item": "https://example.com/electronics/smartphones" } ] } </script>

7. Accessibility Testing

Tekan Tab untuk navigasi keyboard. Screen reader akan membaca "Breadcrumb navigation" dan setiap item.

✅ Accessibility Checklist:

  • ✓ Semantic HTML dengan <nav> dan <ol>
  • ✓ aria-label="Breadcrumb" untuk screen readers
  • ✓ aria-current="page" pada halaman aktif
  • ✓ Keyboard accessible (Tab + Enter)
  • ✓ Focus indicator visible (outline)
  • ✓ Color contrast ratio ≥ 4.5:1 (WCAG AA)
  • ✓ Touch target ≥ 44x44px untuk mobile
